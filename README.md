## 📌 프로젝트 개요

**UniVap**은 대학생들이 함께 식사하는 활동을 통해 새로운 친구를 만들고,  
캠퍼스 안에서 자연스럽게 사회적 관계를 형성하도록 돕는 플랫폼입니다.  
학교 이메일 인증을 통해 신뢰할 수 있는 매칭을 제공하며, 1:1 채팅과 커뮤니티 기능을 통해  
학내 소셜 연결망을 확대하는 것을 목표로 합니다.

## 🛠️ 개발 배경

- 관계 단절: 학업, 취업 준비 등으로 인한 교류 부족
- 거리감 지속: 코로나 이후에도 이어진 대학생 간 거리감
- 개인 성향: 내성적인 학생들의 네트워킹 진입 장벽
- 기존 SNS 한계: 대학생 특화 기능 부족, 진정성 있는 관계 형성 어려움

## 💡 주요 기능

| 기능 | 설명 |
|------|------|
| **대학 인증 기반 로그인** | 학교 이메일 도메인을 통해 재학생 인증을 진행하여 신뢰 기반 매칭 환경을 구축합니다. |
| **1:1 실시간 채팅 기능** | 매칭된 사용자끼리 플랫폼 내에서 안전하고 빠르게 대화할 수 있도록 채팅 기능을 제공합니다. |
| **커뮤니티 게시판 (글 작성, 수정, 삭제)** | 글을 자유롭게 작성하고, 이후 수정 및 삭제도 가능해 개인정보 보호와 피드백 반영이 가능합니다. |
| **프로필 수정 기능** | 닉네임, 프로필 이미지, 자기소개 등을 자유롭게 수정하며 개성과 의도를 표현할 수 있습니다. |

🔐 보안 강화
사용자의 로그인 비밀번호는 bcrypt 알고리즘을 통해 암호화되어 안전하게 저장되며, 민감 정보 보호와 인증 신뢰성을 높였습니다.
