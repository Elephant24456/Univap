# 🍽️ UniVap - 함께 밥 먹는 대학생 네트워킹 플랫폼

> **식사 기반의 자연스러운 만남으로 캠퍼스 내 관계 단절 문제를 해결합니다.**

---

## 📌 프로젝트 개요
<!-- 프로젝트의 목적과 핵심 기능을 간단히 설명 -->
UniVap은 대학생들이 함께 식사하는 활동을 통해 새로운 친구를 만들고,  
사회적 연결과 공동체 의식을 자연스럽게 형성하도록 돕는 플랫폼입니다.  
대학 인증 API를 통해 신뢰 기반의 매칭 서비스를 제공합니다.

---

## 🛠️ 개발 배경
<!-- 어떤 문제를 해결하고자 했는지, 어떤 계기로 시작했는지 서술 -->
- 관계 단절 (학업, 취업 준비로 인한 교류 부족)
- 거리감 지속 (코로나 이후 여전한 거리감)
- 개인적인 성향 (내성적인 성격으로 인한 진입장벽)
- 기존 SNS 한계 (대학생 맞춤 기능 및 신뢰성 부족)

---

## 🎯 기대 효과
<!-- 이 프로젝트를 통해 사용자는 어떤 이득을 얻는가 -->
- 식사 기반 자연스러운 교류 유도
- 고립감 감소 및 소속감 증가
- 대학 인증 기반의 신뢰도 높은 사용자 매칭
- SNS와 차별화된 목적 중심 플랫폼

---

## 💡 주요 기능

| 기능 | 설명 |
|------|------|
| 🎓 **대학 인증 기반 로그인** | 사용자의 학교 이메일 도메인을 기반으로 한 인증 시스템을 통해, **신뢰성 있는 사용자 풀**을 형성합니다. |
| 💬 **1:1 실시간 채팅 기능** | 매칭된 사용자 간 실시간 채팅 기능을 제공합니다. **웹소켓 기반으로 빠른 응답성과 안정성**을 확보했습니다. |
| 🗣️ **커뮤니티 게시판 (글 작성 / 수정 / 삭제)** | 사용자들은 게시판에서 식사 모임, 후기, 자유 주제를 공유할 수 있으며, **자유롭게 수정 및 삭제**가 가능합니다. |
| 🧑‍🎨 **프로필 수정 기능** | 닉네임, 프로필 이미지, 자기소개를 원하는 대로 수정할 수 있어, **자기 표현과 신뢰 형성에 도움**을 줍니다. |
